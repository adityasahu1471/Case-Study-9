# 📘 Course Feedback System

## 📌 Project Overview

The **Course Feedback System** is a Java-based application designed to collect, manage, and analyze student feedback for academic courses. It supports both **console-based execution** and a **modern JavaFX UI dashboard**, enabling students to submit feedback and administrators to monitor results.

## 🏗️ System Architecture

### 📦 Core Classes

* **Admin** → Manages system and reports 
* **Student** → Submits feedback 
* **Faculty** → Represents instructors 
* **Course** → Stores course details & feedback 
* **Feedback** → Stores ratings & comments 
* **Question** → Defines evaluation questions 
* **FeedbackSystem** → Central controller 

---

## 💡 Features

* ✅ Student login & feedback submission
* ✅ Rating system (1–5 scale)
* ✅ Course-wise feedback tracking
* ✅ Admin dashboard (JavaFX UI) 
* ✅ Report generation with average ratings
* ✅ Data validation and error handling
* ✅ Reset and manage feedback system

## ▶️ How to Run

### 🔹 Console Version

1. Compile all `.java` files
2. Run:

   ```bash
   javac *.java
   java Main
   ```

---

### 🔹 JavaFX UI Version

1. Ensure JavaFX is installed
2. Run:

   ```bash
   javac MainApp.java
   java MainApp
   ```

---

## 📊 Sample Workflow

1. Admin initializes system
2. Students are added
3. Courses & questions are created
4. Students submit feedback
5. System calculates averages
6. Admin views reports
